# gym project

> **C언어 기반의 실시간 학교 체육관 운동기구 예약 및 관리 프로그램입니다.** > 아스키 아트(ASCII Art)를 활용한 체육관 도면 시각화와 콘솔 색상 변경, 실시간 시스템 시간 동기화를 통해 사용자가 직관적으로 운동기구의 예약 현황을 파악하고 예약 및 취소를 할 수 있도록 구현했습니다.

---

## important!!!!

* **실시간 예약 및 현황 조회**
  * 현재 시스템 시간(`time.h`)을 반영하여 기구별 이용 시작 시간과 종료 시간을 실시간으로 계산하고 표(`excel()`) 형태로 출력합니다.
* **아스키 아트(ASCII Art) UI 시각화**
  * 체육관 내부의 기구 배치도(`gym()`)를 콘솔에 시각적으로 표현하여 사용자가 기구 번호(1~26번)를 직관적으로 선택할 수 있습니다.
* **실시간 중복 예약 방지 및 예외 처리**
  * 이미 다른 사용자가 이용 중이거나 예약 유효 시간 내에 있을 경우, 중복 예약을 차단하고 재입력을 요구합니다.
  * 1000~4000 사이의 올바른 학번 형식 검사 및 최소 1분 이상의 이용 시간만 입력받도록 예외 처리가 되어 있습니다.
* **간편한 예약 취소 기능**
  * 운동기구 선택 단계에서 `0`번을 입력하면 원하는 기구의 예약을 즉시 취소하고 실시간으로 현황판을 업데이트합니다.

---

## file structure

* `main.c` : 프로그램의 메인 루프가 돌며 예약 여부 확인, 예외 처리, 데이터 업데이트를 담당하는 핵심 소스 코드입니다.
* `my-header.h` : 운동기구 데이터 구조, 아스키 도면 출력, 예약 표 출력, 콘솔 색상 제어 등 UI 및 유틸리티 함수가 포함된 헤더 파일입니다.

---

## date structure

프로그램은 전역 배열과 구조체를 활용하여 운동기구의 상태를 안전하게 관리합니다.

| 변수 / 데이터 구조 | 타입 | 설명 |
| :--- | :--- | :--- |
| `inf[27][2]` | `int` | `[기구 번호][0]`: 예약자 학번 / `[기구 번호][1]`: 이용 예정 시간(분) |
| `T[27][1]` | `int` | `[기구 번호][0]`: 예약 당시의 시스템 기준 환산 시간 (시 * 60 + 분) |
| `tm` | `struct tm` | 현재 실제 PC의 시스템 시간을 받아오는 Windows 시간 구조체 |

## algorithm

1. **대시보드 출력** 회색 텍스트로 체육관 배치도(`gym()`)와 현재 기구별 예약 현황판(`excel()`)을 화면에 시각화합니다.
2. **운동기구 선택** 사용자에게 원하는 기구 번호를 입력받습니다. (`0` 입력 시 즉시 취소 모드로 전환)
3. **중복 예약 검증** 선택한 기구의 학번 데이터가 존재하고, 현재 시간이 기존 예약 종료 시간 이전이라면 "이미 사용 중" 메시지를 띄우고 메인 화면으로 돌아갑니다.
4. **사용자 인증 및 시간 입력** 유효한 학번(1000~4000)과 최소 1분 이상의 사용 시간을 입력받습니다.
5. **예약 완료 및 동기화** 현재 시스템 시간을 기준으로 시작 및 종료 시간을 계산하여 데이터를 저장하고 예약 완료 메시지를 출력합니다.
